Структура файла config.txt (должен лежать в корневой папке проекта)
1)расположение файла с русской локализацией 
например: res\ru.txt
2)расположение файла с английской локализацией
например: res/en.txt
3)расположение файла с русским readme например: res\rus_readme.txt
4)расположение файла с русским aboutme например: res\rus_aboutme.txt
5)расположение файла с английским aboutprog например: res\rus_aboutprog.txt
6)расположение файла с английским readme например: res\eng_readme.txt
7)расположение файла с английским aboutme например: res\eng_aboutme.txt
8)расположение файла с английским aboutprog например: res\eng_aboutprog.txt
расположение файла с предыдущей сессией: data/data.txt
